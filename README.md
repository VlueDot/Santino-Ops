# Santino-Ops

## Project initialization 

```
nvm install 20
nvm use 20
pyenv install
pyenv local $(< .python-version)
npm install -g firebase-tools
firebase login
firebase init functions
cd functions
python -m venv venv 
source venv/bin/activate
pip install nodeenv
nodeenv env
. env/bin/activate
nodeenv -p
pip freeze > requirements.txt 
freeze nrequirements.txt
```
